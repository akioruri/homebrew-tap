# Akioruri Tap

## How do I install these formulae?

`brew install akioruri/tap/<formula>`

Or `brew tap akioruri/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "akioruri/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
